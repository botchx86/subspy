# Security Policy

subspy is a subdomain reconnaissance tool intended for authorized security testing only. See the Disclaimer in [README.md](README.md).

## Reporting a Vulnerability

If you find a security issue in subspy itself (as opposed to a target you're scanning), please report it privately rather than opening a public issue:

- Open a [GitHub security advisory](https://github.com/botchx86/subspy/security/advisories/new) for this repository, or
- Email the maintainer directly (see the GitHub profile for contact info).

Please include:
- A description of the issue and its impact
- Steps to reproduce
- The affected version

We'll acknowledge reports as soon as possible and aim to release a fix before any public disclosure.

## Supported Versions

Only the latest release is supported with security fixes.
